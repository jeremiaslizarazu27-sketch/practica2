# For

Un "for" es como cuando repetís algo muchas veces sin tener que escribirlo todo.
Por ejemplo, si querés contar del 1 al 5, en vez de escribir cada número, el "for" lo hace solo.
Es como decirle a la compu: “hacé esto varias veces”.
## && (Y)
Significa "y".
Se usa cuando tienen que pasar dos cosas al mismo tiempo.
Ejemplo:
Si tengo hambre Y hay comida, entonces como.
Si falta una de las dos, no pasa nada.
## || (O)
Significa "o".
Se usa cuando alcanza con que pase una sola cosa.
Ejemplo:
Si tengo hambre O estoy aburrido, como algo.
Con que una sea verdad, ya alcanza.
## ! (NO)
Significa "no".
Sirve para decir lo contrario de algo.
Ejemplo:
Si NO tengo hambre, no como.
También cambia:
true → false
false → true
