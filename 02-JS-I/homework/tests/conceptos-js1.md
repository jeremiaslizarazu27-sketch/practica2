# Conceptos básicos de JavaScript

## Variables
Una variable es como una caja donde guardamos información.

## Strings
Son textos, como palabras o frases. Siempre van entre comillas.

## Funciones
Son como máquinas que hacen algo cuando las usamos.

## If
Sirve para tomar decisiones.

## Booleanos
Son valores que solo pueden ser true (verdadero) o false (falso).
